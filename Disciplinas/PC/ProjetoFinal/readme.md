# Programação Concorrente

Projeto final da Disciplina <br />

A pesquisa na universidade, em geral, requer uso de equipamentos técnicos para dar suporte ao aprendizado<br />
dos alunos. Em alguns casos, a quantidade de recursos disponíveis são insuficientes para atender a demanda de todos<br />
os alunos de sua graduação. Portanto, o principal objetivo deste trabalho é apresentar um sistema de simulação de acesso ao<br />
laboratório de informática (LINF) do departamento de Ciência da Computação (CiC), UnB. O sistema foi desenvolvido como<br />
sendo o produto de pesquisa e aprendizado sobre programação concorrente.<br />

Para rodar:<br />
gcc ProjetoFinal.c -o simulador -pthread<br />
./simulador<br />
